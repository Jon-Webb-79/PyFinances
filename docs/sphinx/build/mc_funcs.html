Monte Carlo Functions¶

The following classes and functions are used in the Monte Carlo and Date Iteration portion of the PyFinances software suite

MCFunctions¶

The MCFuncs class contains all non Monte-Carlo functions that are used in the date iteration.

Parameters

  • alloc_pay – The paycheck issued to the employee after all deductions.
