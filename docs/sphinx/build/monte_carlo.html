Monte Carlo Functions¶

The functions and classes described in this section are used as the core of the Monte Carlo engine that determines the balance of checking and savings accounts as a function of date.

MoneCarloExec¶

This class contains the functions that operate the Monte Carlo portion of the PyFinances class.

class monte_carlo.MonteCarloExec[source]¶

This class contains functions that are used during the PyFinances Monte Carlo iteration process

determine_pay_allocation(salary: numpy.float32, pay_frequency: str, file_name: str)[source]¶
Parameters

  • salary – The annualy salary before any pay deductions

  • pay_frequency – The frequency of pay allocation, can be WEEKLY, BI-WEEKLY, or MONTHLY

  • file_name – THe name and location of the csv file containing the pay deduction information.

Return deducted_salary

The salary after it has been divided by the pay periods and all deductions have been made

This function determines the actual pay allocation per pay period after deductions have been made.
